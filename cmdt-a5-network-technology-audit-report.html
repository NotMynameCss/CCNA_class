Audit Summary & Recommendations

  • Strong dual-core / dual-distribution design: HQ-MLSW1↔MLSW2 LACP EtherChannel + 3 HSRP-paired distribution pairs cover VLAN 10/20/30/70/80/90.
  • Comprehensive OSPF single-area deployment (process 15) with redistribution and default-information originate from Gate-HQ edge.
  • Dual ASA 5506-X firewall pair with FW_LINK interconnect, security levels, inspection policies, and named ACLs — solid perimeter design.
  • Dual DHCP servers (10.200.200.5/6) with redundant ip helper-address on all SVIs; dedicated DNS server (10.200.200.7).
  • PortFast + BPDU Guard on all access ports; SSH v2 with ACL-restricted VTY; password encryption on managed switches.
  • MISSING: HSRP Preempt & Tracking — without preempt, the active router does not recover automatically after failover.
  • MISSING: Route Summarization — no area range or summary-address; every /24 and /30 floods the OSPF LSDB.
  • MISSING: DHCP Snooping + DAI + IP Source Guard — Layer 2 security hardening absent on all access switches.
  • MISSING: QoS — no traffic classification or DSCP marking; critical for real-world enterprise with voice/video.
  • MISSING: IPv6 — fully disabled everywhere; no dual-stack, SLAAC, DHCPv6, or OSPFv3.
  • MISSING: VPN completion — IPsec IKE/crypto map configured on Gate-HQ but marked Incomplete; ACL 150 undefined; no GRE tunnel interface.
  • MISSING: Rapid PVST+ — topology uses classic PVST+ (slower convergence); upgrade to rapid-pvst recommended.
  • MISSING: Root Bridge priority configuration — no explicit spanning-tree priority set; STP root election is non-deterministic.
  • MISSING: Syslog / NTP / SNMP — no centralized logging, time synchronization, or network monitoring configured.
